Student Life Management System

Overview

This project is a static HTML-based Student Life Management System designed to help students organize academic tasks, schedules, and deadlines. It demonstrates foundational web development concepts using structured layout and basic interface design.

Purpose

The goal of this project is to apply first-semester web development knowledge to create a simple and functional student productivity interface.

Features

Task and assignment display

Structured layout for schedules

Clean and organized interface

Easy navigation within the page
